## Por que versionar o código de seu projeto?

Versionamento é algo importantíssimo quando estamos falando de desenvolvimento de software.

Version control is a very crucial part of software development. A well organized version control makes it easy to compare files, perform merges (or emergency rollbacks) into the production environment.
